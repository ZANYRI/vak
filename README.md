# Тестовые промпты

Это тестовые промпты для систем.

Ссылки на промпты:

- [service.md](service.md)
- [site.md](site.md)
- [3DBenchy.md](3DBenchy.md)

Краткое описание каждого промпта:

- `3DBenchy.md`: Анимированный self-contained SVG с псевдо-3D моделью 3D Benchy; без JS и внешних зависимостей, с волнами и плавной раскачкой.
- `service.md`: Спецификация бэкенд-сервиса на Go для управления подписками и биллингом (Postgres, очередь, JWT, расчёт счетов, Docker).
- `site.md`: Техническое задание для двуязычного (RU/EN) рецептурного веб-сайта на Next.js с i18n, адаптивным UI и Docker-deploy.
